Проект игры "Угадай число".

Проект выполнен максимально просто со множеством допущений.
Главная задача - в легком виде продемонстрировать понимание принципов SOLID.
В коде проекте в комментариях указано где эти принципы встречаются и как они соблюдаются.

1. Принцип SRP продемонстрирован на примере типа UserController.
2. Принцип OCP продемонстрирован на примере метода UserValidate внутри класса UserController.
3. Принцип LSP продемонстрирован на том же примере типа UserController.
4. Принцип ISP продемонстрирован на примере абстракции IValidate.
5. Принцип DIP продемонстрирован на примере класса ControllerBase.
